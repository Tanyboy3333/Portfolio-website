Pancreatic Cancer Detection

This project implements a CNN-based deep learning model to detect pancreatic cancer from CT scan images. The trained model (cancer_model.h5) is served using a Flask backend that preprocesses uploads and returns real-time predictions. A web interface built with HTML, Bootstrap, and JavaScript enables users to easily upload scans and view results. Deployment is automated through GitHub Actions CI/CD and hosted on Render for public access.

Sentiment Analysis Chatbot

This project implements a hybrid chatbot for sentiment detection using both a VADER lexicon-based model and a Logistic Regression classifier trained on TF-IDF features. It preprocesses user text (removing URLs, special characters, stopwords, and applying lemmatization) before generating predictions. A Gradio-powered interface provides real-time interaction, combining outputs from both models for robust sentiment classification (positive, neutral, negative). The app is deployed on Hugging Face Spaces for global accessibility.

DataDoc AI – AI-Powered Document Q&A Chatbot

DataDoc AI – AI-Powered Document Q&A Chatbot is a web application that lets users upload documents in multiple formats (PDF, DOCX, TXT, CSV, images, spreadsheets, etc.) and interact with them through a conversational chatbot powered by Large Language Models. The system uses LlamaParse to extract content, Cohere embeddings and LlamaIndex to build a semantic index, and a fast Groq LLM for generating contextually accurate answers. A sleek Gradio interface provides real-time Q&A with streaming responses, while deployment on Render.com makes the app globally accessible.
